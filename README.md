## Enunciado

Escreva um programa que leia dois números inteiros e determine o menor múltiplo entre eles. O menor múltiplo é o menor número positivo que é divisível por ambos os números. Considere que ambos os números serão sempre diferentes e maiores que zero.

Exemplo:
Entrada:
4
6

Saída:
12
